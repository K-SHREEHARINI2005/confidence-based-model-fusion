# confidence-based-model-fusion
A Java-based implementation of Confidence-Based Model Fusion (CBMF) for robust adversarial detection in multi-controller Software-Defined Networks (SDNs), inspired by recent research in ML-based network security.

Project Overview
*Software-Defined Networks (SDNs) are vulnerable to adversarial attacks targeting ML-based Intrusion Detection Systems.
*This project proposes a Confidence-Based Model Fusion (CBMF) approach where multiple models collaborate, and decisions are fused using confidence values to achieve robust detection.

Key Concepts
*Software-Defined Networking (SDN)
*Multi-Controller Architecture
*Machine Learning Based IDS
*Adversarial Attack Detection
*Model Confidence Scoring
*Model Fusion Strategy

Objectives
✔ Improve adversarial detection accuracy
✔ Increase robustness of ML-based IDS
✔ Combine multiple model predictions
✔ Use confidence scores for decision fusion
✔ Enhance network security in SDNs

Features
*Confidence-based decision fusion
*Multi-model detection system
*Adversarial attack simulation support
*Java implementation for experimentation
*Modular architecture for future extension

Technologies Used
*Java
*Machine Learning Concepts
*Software-Defined Networking (SDN)
*Network Security Principles

Working Principle
*Multiple detection models analyze network traffic.
*Each model produces a prediction with confidence score.
*Confidence-Based Model Fusion combines outputs.
*Final decision is generated for attack detection.
*System improves robustness against adversarial inputs.
